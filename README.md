# Metasploitable Penetration Testing Lab

## Overview

This project demonstrates a basic penetration testing workflow using Kali Linux against the Metasploitable vulnerable machine.

The lab covers:

- Network reconnaissance
- Port scanning
- Service enumeration
- Exploitation using Metasploit Framework

## Tools Used

- Kali Linux
- Nmap
- Metasploit Framework

## Target Machine

Metasploitable 2

## Lab Steps

1. Host discovery
2. Port scanning
3. Service enumeration
4. Exploitation
5. Post-exploitation

## Disclaimer

This lab was performed in a controlled environment using intentionally vulnerable machines for educational purposes only.
